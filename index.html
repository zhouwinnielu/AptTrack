AptTrack

A static tracker for live 1B/1B availability at Avalon Princeton Circle. GitHub Actions scrapes the source listing data twice a day and republishes this page with updated price history.

GitHub Actions runs in UTC, so this project uses an hourly workflow plus an `America/New_York` time check to keep the actual scrape times aligned to 9:00 AM and 5:00 PM Eastern through daylight saving changes.
